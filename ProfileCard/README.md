# Social Links Profile UI

A simple and responsive Social Links Profile card built using HTML and CSS. This project displays a personal profile with social media links in a clean and modern card layout.

## 📸 Preview

The UI contains:

- Profile Image
- Name
- Location
- Professional Tagline
- Social Media Buttons
  - GitHub
  - Frontend Mentor
  - LinkedIn
  - Twitter
  - Instagram

## 🚀 Technologies Used

- HTML5
- CSS3
- Flexbox
